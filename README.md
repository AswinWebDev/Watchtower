# OneChain Spend Guardian (Watchtower)

## What is Watchtower?
Watchtower is an **AI-powered autonomous agent and smart-vault** built on OneChain's Move infrastructure. It solves a massive pain point for GameFi and DeFi users: preventing wallet drain, overspending, and interaction with malicious smart contracts.

Instead of relying on manual discipline or high technical literacy, users simply speak to Watchtower in natural language (e.g., *"Max 30 USDO per week on any GameFi dApp"*, or *"Block new contracts with less than 100 unique users"*).

The Watchtower AI Agent parses these human intentions into structured on-chain JSON policies and writes them to a secure `guardian_vault.move` smart contract. All funds deposited in the vault are then protected by these hardcoded, on-chain rules. 

## OneHack 3.0 Alignment
Built for the **OneHack 3.0 AI Track** (with strong GameFi crossover utility):
1. **Purposeful AI:** We don't just use an LLM for a chatbot. We use an LLM (Venice AI) as an autonomous compiler—translating human language into formal mathematical policies that directly govern fund transfers.
2. **Move Infrastructure:** The system relies entirely on a custom Move smart contract to serve as the secure intermediary vault. All rules generated by the AI are permanently enforced at the consensus layer.
3. **OneWallet Integration:** Native, seamless connection directly to the browser extension via the `@onelabs/wallet-standard` API.

## Architecture Architecture

The application is built in three layers to provide a true Enterprise-Grade MVP:

1. **The Brain (Node.js/Express Backend):** 
   - Receives human-language rules from the frontend.
   - Interfaces with an LLM (llama-3.3-70b via Venice AI) to parse intent.
   - Compiles intent into a strict, validated JSON structure.
2. **The Vault (OneChain Move Contract):**
   - `guardian_vault.move` acts as a proxy wallet.
   - Stores the structured policies on-chain.
   - Intercepts and validates transactions before forwarding them to games/dApps.
3. **The Command Center (React Frontend):**
   - A premium, glassmorphism UI built with Vite and Tailwind.
   - Native OneWallet extension connection.

## Quick Start

### 1. Prerequisites
- Node.js (v18+)
- OneChain CLI installed locally
- OneWallet Chrome Extension installed

### 2. Run the AI Backend
```bash
cd server
npm install
# Create a .env file and add your VENICE_API_KEY
npm run dev
```

### 3. Run the Frontend Command Center
```bash
cd Watchtower
npm install
npm run dev
```

### 4. Deploy the Smart Contract (Testnet)
```bash
cd contracts
onechain move build
onechain client publish --gas-budget 100000000
```
